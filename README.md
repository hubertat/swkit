# swkit
swkit - HomeKit enabled switch/input/roller shutter switch, monitor

## features
* HomeKit enabled (github.com)
* switch/button input
* light output
* input - light output relation
* outlet output
* thermostat output
* influx sensor (temperature for thermostat)
